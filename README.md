# PUCRS-Algoritmos-e-Estruturas-de-Dados-II

Repositório destinado às atividades desenvolvidas durante a disciplina **Algoritmos e Estruturas de Dados II**, cursada no 4º semestre da graduação em Engenharia de Software (PUCRS).

## 🎯 Objetivo da disciplina

Aprofundar os conhecimentos em algoritmos e estruturas de dados, com foco em organização, eficiência e manipulação de dados, utilizando conceitos fundamentais para resolução de problemas computacionais.

---

## 🧩 Estrutura do repositório

* **Atividades de aula**: exercícios práticos desenvolvidos ao longo do semestre para fixação dos conteúdos abordados.
* **Trabalho 1 (T1)**: projeto avaliativo intermediário.
* **Trabalho 2 (T2)**: projeto final da disciplina.
